# Input 
The app doesn’t ask the user to type in any information.All the input is already written into the React Native code.It includes the student’s name, the program they’re studying, and the name of their school.These details are sent to three Text components when the app starts up.

# Process
The app uses one main View component to hold everything and three Text components to show the information.
StyleSheet is used to style the screen, setting the background to yellow, centering the text both vertically and horizontally, formatting the text, and adding space between each line.Expo starts the development server, and React Native loads the component structure and styles before displaying the app.

# Output  
The result is a simple mobile screen with a yellow background.
The student’s name, degree program, and school name are shown in centered black text.This example shows how React Native components and styling work, along with the input-process-output approach.
