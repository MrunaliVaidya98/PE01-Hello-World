# Input

The application does not require interactive user input. The three values shown on the screen—my name, degree program, and school—are entered directly in the source code as text. When the application launches, React Native reads these predefined values and makes them available to the user-interface components.

# Process

The program uses one View component as the main container and three Text components to display the required information. StyleSheet creates the visual design by setting the background to yellow, centering the content vertically and horizontally, adding spacing, and formatting the text. Expo starts the development server, and React Native converts the component structure into a mobile interface that can run in Expo Go.

# Output

The application displays a single mobile screen with a yellow background. Centered on the screen are three text messages showing my name, degree program, and school. The final result demonstrates basic React Native component structure, styling, and rendering while meeting the requirement to use one View, three Text components, and StyleSheet.
